# AWS Architecting - Formação em Arquitetura de Soluções AWS


## Sobre o Curso

Bem-vindo ao repositório oficial do curso **AWS Architecting**, promovido pela **Impacta Tecnologia**.

Este treinamento foi desenvolvido para capacitar profissionais de tecnologia a projetar, implementar e avaliar arquiteturas modernas na nuvem utilizando os principais serviços da Amazon Web Services (AWS).

O curso apresenta uma abordagem prática e arquitetural, indo além da utilização de serviços individuais, preparando o aluno para pensar como um **Arquiteto de Soluções Cloud**.

---

# Objetivo

O objetivo deste curso é desenvolver a capacidade de:

- Projetar arquiteturas escaláveis, resilientes e seguras na AWS;
- Escolher corretamente serviços AWS conforme requisitos técnicos e de negócio;
- Aplicar boas práticas de arquitetura em ambientes reais;
- Avaliar custos, desempenho, segurança e confiabilidade;
- Tomar decisões arquiteturais utilizando frameworks profissionais.
